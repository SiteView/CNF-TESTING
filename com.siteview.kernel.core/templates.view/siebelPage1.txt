<h3>Use the following steps to setup SiteView monitoring for Siebel:</h3>
 <ol>
 <li><b>First, install the Siebel Server Manager client</b> on the machine where SiteView is running</li>
 <li>Use the wizard to tell SiteView <b>how to connect to the Siebel server</b> (this page)</li>
 <li>Give a <b>name for the monitor groups</b> to be created for the SiteView Siebel monitoring</li>
 <li><b>Specify application information</b> to create the server and system monitors to watch key server performance metrics</li>
 <li>Reuse the wizard for another server or exit to the SiteView main page to see the monitor results</li>
 <li>After exiting the wizard, <b>setup JDBC connectivity</b> to the database used by Siebel and create a 
<b>SiteView Database Monitor</b> to check Siebel database connectivity</li>
 </ol>
